# Personal assistant app for students

classes timings, assignment deadlines, quizes dates and timings, links for submissions, links for classes etc.   ..... headache
In the current scenario of online classes ... different teachers use different apps for the online meeting ,paper evaluation and homework submission..
A student always faces  the difficulty to properly manage the class schedule, assignment submissions, exams...
So here is a third party personal assistant app that can handle all this just in one place.
Automatically updates with link and timings whenever new meeting, assignment, exam or anything is scheduled and much more.
